# Agentic Visualization 3D Test

## Introduction

This is a test markdown file to demonstrate the 3D visualization capabilities.

## Features

### 2D Mode
- Traditional D3.js force-directed graph
- SVG-based rendering
- Interactive zoom and pan

### 3D Mode
- WebGL-based rendering
- Spatial graph visualization
- Orbit controls for navigation
